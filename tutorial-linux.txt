
====================
1) Instalar composer siguiendo los pasos (https://getcomposer.org/download/)
2) Ejecutar: sudo apt install php-xmlwriter (https://stackoverflow.com/questions/6384979/how-to-enable-xmlwriter-after-php-having-been-compiled)
3) Ejecutar: composer install
========================

Para entender el block.json de la carpeta src/blocks/header/block.json :
https://developer.wordpress.org/block-editor/reference-guides/block-api/block-registration/

Para entender el codigo src/clocks/header/index.ts :
https://codigoconjuan.com/creando-tu-primer-bloque-de-gutenberg-con-wp-scripts-javascript-y-react/

La parte frontal se renderiza desde templates/blocks/header/markup.php, no desde el metodo save.tsx de src/blocks/header/save.tsx :
https://developer.wordpress.org/block-editor/how-to-guides/block-tutorial/creating-dynamic-blocks/
https://medium.com/nicasource/how-to-build-a-custom-dynamic-gutenberg-block-step-by-step-a345e97aee98

=========================

Para el desarrollo ejecutar:
npm run start:hot
